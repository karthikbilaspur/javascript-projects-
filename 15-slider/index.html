HTML: The Structure

HTML defines the structure and meaning of web content. HTML5 introduced semantic tags like <article> and native <video> elements.

CSS: The Style

CSS controls presentation. Flexbox and Grid revolutionized layouts. CSS variables and media queries enable responsive design without frameworks. 

.container {
  display: grid;
  grid-template-columns: 1fr 3fr;
  gap: 1rem;
  --main-color: #4f46e5;

JavaScript: The Behavior

JS makes pages interactive. ES6+ brought arrow functions, promises, and modules. It runs in browsers and on servers via Node.js.

HTTP: The Protocol

HTTP is the foundation of data communication on the web. Methods like GET, POST, PUT, DELETE define CRUD operations. HTTP/2 and HTTP/3 improved speed with multiplexing and QUIC. 

GET /api/users HTTP/1.1
Host: example.com
Accept: application/json

Git: Version Control

Git tracks changes in source code. Branching, merging, and pull requests enable team collaboration. Distributed architecture means every clone is a full backup. 

git checkout -b feature/login
git add.
git commit -m "Add login form"
git push origin feature/login

React: UI Library

React uses components and a virtual DOM for efficient UI updates. Hooks like useState and useEffect replaced class components. Unidirectional data flow makes apps predictable. 

function Counter() {
  const [count, setCount] = useState(0);
  return <button onClick={() => setCount(count + 1)}>
    {count}
  </button>;
}
