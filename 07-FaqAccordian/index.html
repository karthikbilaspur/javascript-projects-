HTML, CSS & JavaScript FAQ

Common questions answered about the core web technologies

HTML is the structure - it defines elements like headings, paragraphs, and images using tags. CSS is the style - it controls colors, layout, fonts, and spacing. JavaScript is the behavior - it makes pages interactive, handles events, and updates content dynamically.

Every HTML element is a box made of 4 layers: content, padding, border, and margin. By default, width and height only set the content size. Use box-sizing: border-box; to make width/height include padding and border, which makes layouts much easier.

var: Function-scoped, hoisted, and can be redeclared. Avoid in modern JS.
let: Block-scoped, can be reassigned but not redeclared in the same scope.
const: Block-scoped, cannot be reassigned. Use by default, switch to let only if you need to reassign.

Depends on what you're centering:

  • Horizontally: margin: 0 auto; for block elements with a set width
  • Both directions: Use Flexbox - display: flex; justify-content: center; align-items: center;
  • Both directions: Use Grid - display: grid; place-items: center;

Instead of adding listeners to every child element, you add one listener to a parent. When an event bubbles up, you check event.target to see which child fired it. This is faster and works for elements added to the DOM later. 

document.querySelector('ul').addEventListener('click', e => {
  if(e.target.tagName === 'LI') {
    console.log('Clicked:', e.target.textContent);
  }
});
