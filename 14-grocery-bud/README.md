# Lorem Ipsum Shopping List Generator

A vanilla JavaScript mini project that generates fake shopping list items with Lorem Ipsum names, descriptions, prices, and placeholder images. Built as an extension of the classic "Grocery Bud" CRUD app.

## ✨ Features

 Add custom items: Type a product name or use the "Generate Lorem Product" button
 Autogenerated details: Each item includes:
   Random Lorem product name
   2sentence Lorem Ipsum description
   Random price between $10–$99
   Unique placeholder image from `https://picsum.photos`
 Full CRUD: Add, edit, delete individual items, or clear all
 Persistent storage: Saves your list to `localStorage` and reloads on refresh
 Responsive UI: Card layout works on mobile and desktop
 Instant feedback: Success/error alerts for all actions

## 🛠️ Tech Stack

 HTML5  Semantic structure
 CSS3  Flexbox, custom properties, responsive design
 Vanilla JavaScript  DOM manipulation, localStorage, event handling
 Font Awesome  Edit/delete icons
 Picsum Photos  Random placeholder images

### 🚀 How To Run

1. Clone or download this folder
2. Open `index.html` in your browser
3. No build tools or npm needed!

### 🎯 How To Use

1. Add manual item: Type in input → click "Add Item"
2. Generate Lorem item: Click "Generate Lorem Product" for instant fake product
3. Edit: Click "edit" button → update name in input → click "edit" again
4. Delete: Click "delete" to remove single item
5. Clear all: Removes entire list from UI + localStorage

### 💡 Key Concepts Used

| Concept | Implementation |
|  |  |
| CRUD Operations | Create/read/update/delete items in DOM + localStorage |
| Object Data Model | Items stored as `{id, name, desc, price, img}` |
| localStorage API | `JSON.stringify` / `JSON.parse` for persistence |
| Template Literals | Dynamic HTML generation for each card |
| Event Delegation | Edit/delete listeners added per item |
| External API | `picsum.photos/seed/{id}` for unique images |
| Randomization | `Math.random()` for names, prices, descriptions |

## 📝 Notes

 Images are hotlinked from `picsum.photos` and require internet
 Edit function currently only updates the name field for simplicity
 IDs use `Date.now()` for uniqueness
