What is Semantic HTML?

Semantic tags like <header>, <main>, <article>, and <nav> describe their meaning to browsers and assistive tech.

  • SEO: Search engines understand content hierarchy better
  • Accessibility: Screen readers can navigate by landmarks
  • Maintainability: Code is self-documenting
<article>
  <h2>Blog Post Title</h2>
  <p>Content goes here...</p>
</article>

Flexbox vs Grid

Flexbox: 1-dimensional. Great for nav bars, centering, distributing space in a row/column.

Grid: 2-dimensional. Perfect for full page layouts, card grids, complex alignment.

CSS Specificity

Calculated as: inline > ID > class/attribute > element. Avoid !important unless necessary.

#nav.button.active { } beats .button { }

Event Loop Basics

JS is single-threaded. The event loop handles async tasks: Call Stack → Web APIs → Callback Queue → Event Loop pushes to Stack when empty.

Promises vs Async/Await

// Promises
fetch('/api').then(res => res.json()).then(data => console.log(data));

// Async/Await - cleaner
async function getData() {
  const res = await fetch('/api');
  const data = await res.json();
  console.log(data);
}
