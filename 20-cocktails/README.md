# Mixology Hub

A responsive cocktail recipe website built with vanilla HTML, CSS, and JS.

## Features

- Search 600+ cocktails from TheCocktailDB
- Filter by ingredient, category, alcoholic/non-alcoholic  
- View detailed recipes with measurements
- Save favorites locally
- Fully responsive design

## Setup

1. Download all 4 files into the same folder
2. Open `index.html` in your browser
3. That's it - no build step needed
